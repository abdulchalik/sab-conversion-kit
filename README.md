# SAB Conversion Kit

The SAB Conversion Kit is a Tailwind CSS-based starter kit for building landing pages and high-converting UI components.

## 🚀 Features

- Tailwind CSS pre-configured
- SAB Brand Colors & Fonts
- Reusable Components
- Clean folder structure
- Ready for landing page & marketing pages

---

## 📦 Installation

Clone repo:

```bash
git clone https://github.com/abdulchalik/sab-conversion-kit.git
cd sab-conversion-kit
```

Install dependencies:

```bash
npm install
```

## 🛠️ Development

Run the Tailwind watcher:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```
