Cara Pakai — Integration Guide

Setiap component bisa dipakai dengan tiga cara:

  1. Standalone — buka file langsung di browser, lihat component bekerja.
  2. Copy-paste HTML block — salin section HTML yang diberi komentar ke halaman Anda, bawa serta <style> dan <script>-nya.
  3. Include / iframe — embed sebagai iframe untuk preview cepat dalam builder.
<!-- Contoh integrasi pricing-table.html ke halaman Anda -->

<!-- 1. Tambahkan Google Fonts di <head> -->
<link href="https://fonts.googleapis.com/css2?family=Open+Sans:wght@400;600;700;800&display=swap" rel="stylesheet" />

<!-- 2. Tambahkan Tailwind CDN di <head> (jika belum ada Tailwind) -->
<script src="https://cdn.tailwindcss.com"></script>

<!-- 3. Copy blok <style> dari component ke <head> halaman Anda -->
<!-- 4. Copy blok HTML component ke <body> -->
<!-- 5. Copy blok <script> dari component ke akhir <body> -->
<!-- 6. Edit data/config di dalam <script> (WA_NUMBER, PLANS, dll) -->
